## miscellaneous notes

```sh
bun init
```

```sh
bun install # instead of npm install
```

```sh
bun run index.ts
```

```bash
bun run --watch index.ts
```

```bash
bun test # will automatically run [...].test.tsx files
```

```bash
bunx cowsay -d 'hi' # instead of npx cowsay -d 'hi'
```
